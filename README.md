## 📦 Install

```bash
pip install zeroskids
```

## 🔧 Usage

```python

original_code = "print('skidzz supremacy')"
obfuscated = obfuscate(original_code)

print(obfuscated)
```

```python

```

✅ works on `.py`, `.js`, `.ts`
